En MOVA hay definida una vista para mostrar la información sobre protección de datos. Por defecto viene con textos de prueba que hay que cambiar.

Hay dos secciones, la protección de datos resumida y la completa:

data-protection-view.html

        </strong>
      </mv-card-title>
    </mv-card-header>

    <mv-card-content>
      ...
      // seccion resumida
      ...

      <mv-button
        color="primary"

        appearance="raised"
