Información
Información del componente mv-table-responsive.
code
El componente mv-table-responsive se basa en el componente mat-table de Angular Material, al que se le añade funcionalidad y estilos adicionales.

Ejemplo de uso
Uso básico del componente mv-table-responsive.
code
A continuación se muestra un ejemplo básico de uso del componente.

HTML
TS

  <!-- Ejemplo con datos en servidor -->
  <mv-table-responsive
    *ngIf="loadDataSource"
    [dataSourceLoader]="loadDataSource"
    [pageSize]="6">
  </mv-table-responsive>

  <!-- Ejemplo con datos en local -->
  <mv-table-responsive
    *ngIf="dataStudent"
    [data]="dataStudent"
    [pageSize]="6">
  </mv-table-responsive>

dataSourceLoader
Datos cargados desde servidor.
code
La propiedad dataSourceLoader permite cargar los datos desde el servidor. Además de la propiedad, hay que definir lo siguiente:

Un servicio rest que obtenga los datos deseados

Una clase que implemente mvLoadData, que servirá de interfaz entre el servicio y el values-list

En el código asociado al values-list, se debe crear un objeto de la clase LoadData, al que se le debe pasar el servicio que obtiene los datos

Si se quiere definir un paginador no se hará con la propiedad pageSize, sino pasando el parámetro al servicio que obtiene los datos
