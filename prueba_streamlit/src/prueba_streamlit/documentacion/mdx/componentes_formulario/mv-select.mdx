Información
Información del componente mv-select.
code
El componente mv-select se basa en el componente mat-select de Angular Material, al que se le añade funcionalidad y estilos adicionales.

Ejemplo de uso
Uso básico del componente mv-select.
code
A continuación se muestra un ejemplo básico de uso del componente.

HTML
TS

  <mv-select-x
    [(ngModel)]="select001Model"
    [options]="selectOptionsEstaciones">
  </mv-select-x>

ariaLabel
Etiqueta para accesibilidad
code
La etiqueta ariaLabel se utiliza para proporcionar una descripción de texto alternativa para elementos que no tienen un texto visible por si mismo y aseguarse que usuarios con lectores de pantalla y otros asistentes de accesibilidad puedan entender el propósito del componente.

HTML
TS

    <mv-select-x
      arialLabel="Tu texto personalizado">
    </mv-select-x>
