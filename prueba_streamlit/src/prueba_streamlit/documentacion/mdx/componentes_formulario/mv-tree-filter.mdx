Información
Información del componente mv-tree-filter.
code
Este componente sirve para mostrar un listado, en forma de categorías y subcategorías con un buscador que permite filtrar entre todos los elementos.

Al pulsar en un elemento padre, se expandirán los elementos hijo. Al pulsar en un elemento hijo, se abrirá la url que esté configurada para ese elemento.

treeFilterItems
Datos que se mostrarán en el componente.
code
La propiedad treeFilterItems define los datos que muestra el componente con sus respectivos niveles y características.

Los datos deben estar definidos como un JSON con los siguientes atributos:

title: título del elemento
subtitle: subtítulo del elemento
link: comportamiento al pulsar el elemento
type: puede tomar los valores url o state dependiendo si queremos navegar a una url externa o a un estado de la app
uri: url o estado de la app donde queremos navegar
children: elementos del nivel hijo
css: Clase css para sobrescribir el estilo de los elementos

TS
Array TS

  <!-- Ejemplo con JSON definido en el código -->

    [treeFilterItems]='exampleCountries'>
