El componente mv-button se basa en el componente mat-button de Angular Material, al que se le añade funcionalidad y estilos adicionales.

Ejemplo de uso
Uso básico del componente mv-button.
code
A continuación se muestra un ejemplo básico de uso del componente.

HTML
TS

  <mv-button>
    Botón básico
  </mv-button>

  <mv-button
    appearance="raised"
    color="primary">
    Acción
  </mv-button>

  <mv-button
    size="small"
    appearance="fab">
    Fab
  </mv-button>

code

    appearance="raised"
    adapt=false>

  </mv-button>

clickEvent
Evento click del botón.
code

HTML
TS

  <mv-button
    appearance="raised"

ariaLabel
Etiqueta para accesibilidad
code
La etiqueta ariaLabel se utiliza para proporcionar una descripción de texto alternativa para elementos que no tienen un texto visible por si mismo y aseguarse que usuarios con lectores de pantalla y otros asistentes de accesibilidad puedan entender el propósito del componente.

HTML
TS

  <mv-button
    arialLabel="Tu texto personalizado">
  </mv-button>
