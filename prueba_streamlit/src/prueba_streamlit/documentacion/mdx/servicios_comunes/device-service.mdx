Información
Información del servicio device-service.
code
Este servicio proporciona funcionalidad relacionada con las propiedades del dispositivo en el que se ejecuta la app.

getMobileInfo
Información del dispositivo
code
Devuelve información del dispositivo movil.

Valores que devuelve:

cordovaVersion: Versión de cordova

isSimulator: Si está siendo ejecutado en un simulador

model: Modelo del dispositivo

manufacturer: Compañía que manufacturó el dispositivo

serialNumber: Nº de serie del dispositivo

uuid: UUID del dispositivo

platform: Sistema operativo del dispositivo

version: Versión del sistema operativo del dispositivo

getWebInfo
Información del navegador.
code
Devuelve la información del navegador

Valores que devuelve:

browserCodeName: Contiene el nombre código del navegador

browserEngine: Contiene el número de compilación del navegador

browserName: Contiene un DOMString con el nombre oficial del navegador

browserPlatform: Contiene el nombre de la plataforma del navegador

browserVersionInfo: Contiene la versión del navegador

browserUserAgent: Contiene la cadena del userAgent

isCookieEnabled: Indica si el navegador tiene activadas las cookies

language: Contiene una cadena representando el idioma del navegador

platform: Plataforma del navegador

isMobileApp
Indica si es una aplicación movil.
code
Devuelve un boolean a true indicando si la aplicación se está ejecutando en un dispositivo móvil. Si devuelve False, se esta ejecutando en un navegador.
