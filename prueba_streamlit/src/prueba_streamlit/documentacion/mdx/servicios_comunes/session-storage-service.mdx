Información
Información del servicio session-storage-service.
code
Este servicio sirve de interfaz para acceder al sessionStorage del navegador. Se puede usar en web y móvil.

init
Resetear el nodo principal.
code
El evento init permite resetear el nodo principal de sessionStorage donde se almacena la información de MOVA.

init(reset)

reset

true: resetea el nodo principal de sessionStorage
false: resetea el nodo principal de sessionStorage si está creado y lo crea en caso contrario

  <p>
    <strong>
      Resetear el nodo principal de mova mediante el evento init:
    </strong>
  </p>
  <mv-button
    color="primary"
    (clickEvent)="initEvent()">
    init(true)
  </mv-button>

    color="primary"
    (clickEvent)="getItemsExample()">
    getItems()
  </mv-button>

  <p><strong>Nodo principal de MOVA en sessionStorage:</strong></p>
  <p>{{movaSessionStorageNode | json}}</p>

setItem
Almacenar información en sessionStorage.
code
El evento setItem permite almacenar información en el nodo principal de sessionStorage donde se almacena la información de MOVA.

setItem(key, value, crypt)

key

'string': cadena con la clave del objeto que estamos almacenando
value

'string': valor que estamos almacenando
crypt

true: encripta el valor que estamos almacenando
false: no encripta el valor que estamos almacenando

getItem
Recuperar un objeto concreto de sessionStorage.
code

getItem(key)

key

'string': cadena con la clave del objeto que queremos recuperar
