Información
Información del componente mv-viewport-info.
code
Este componente sirve para mostrar la información del viewport en tiempo real.

Ejemplo de uso
Ejemplo de uso del componente mv-viewport-info.

    *ngIf="showViewport1">
  </mv-viewport-info>
