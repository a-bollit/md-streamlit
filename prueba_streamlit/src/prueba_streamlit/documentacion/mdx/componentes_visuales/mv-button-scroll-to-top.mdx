Información
Información del componente mv-button-scroll-to-top.
code
El componente mv-button-scroll-to-top sirve para disponer de un botón anclado en la esquina inferior derecha de la pantalla, que haga scroll y nos dirija a la parte superior de la pantalla en caso de pulsar sobre él. Se puede configurar para que haga scroll a otra parte específica que definamos, o que el scroll sea hacia abajo.

  </mv-button-scroll-to-top>

fontSet
Familia de iconos para login top.
code
La propiedad fontSet permite definir la familia de iconos que queremos usar, tan solo en caso de querer usar Font Awesome. Si queremos usar los iconos de Material no hay que usar esta propiedad.

HTML
TS

  <mv-button-scroll-to-top
    fontSet="far"
    icon="fa-address-book">
  </mv-button-scroll-to-top>

  <mv-button-scroll-to-top
    fontSet="fas"
    icon="fa-address-book">
  </mv-button-scroll-to-top>

  <mv-button-scroll-to-top
    icon="build">
  </mv-button-scroll-to-top>
