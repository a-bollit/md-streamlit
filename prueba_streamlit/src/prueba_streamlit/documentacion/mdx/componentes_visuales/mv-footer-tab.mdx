Información
Información del componente mv-footer-tab.
code
El componente mv-footer-tab se basa en el componente mat-tab de Angular Material, al que se le añade funcionalidad y estilos adicionales.

Sirve para organizar la información y secciones de una app

Ejemplo de uso
Uso básico del componente mv-footer-tab.
code
Este componente sirve para poner una barra fija en la parte de abajo de la app movil de hasta 6 iconos que pueden servir de menú o solapas.

Para ver ejemplos más realistas y completos del componente ver en los ejemplos de maquetación

A continuación se muestra un ejemplo básico de uso del componente.

HTML
TS

  <mv-footer-tab
    [tabItems]="arrayTabItems">

    <div content1>
      Contenido1
    </div>
    <div content2>
      Contenido2
    </div>
    <div content3>
      Contenido3
    </div>
  </mv-footer-tab>

selectedItem
Posición seleccionada.
code
La propiedad selectedItem permite seleccionar la vista que queremos mostrar de inicio, así como en cualquier momento cambiar la vista a mostrar cambiando el valor de la variable. También nos permite observar el valor de la vista seleccionada.

Si no especificamos ningún valor, se mostrará la vista correspondiente a la primera posición.

HTML
TS

  <p><strong>Con selectedItem seleccionado inicialmente a 2:</strong></p>

  <mv-footer-tab
    color="primary"
    [(selectedItem)]="selectedItemExample"
    [tabItems]="arrayTabItems3">

    <div content1>
      Contenido1
    </div>
    <div content2>
      Contenido2
    </div>
    <div content3>
      Contenido3
    </div>
  </mv-footer-tab>

  <div class="buttons">
    <mv-button
      (clickEvent)="selectItem(0)"
      appearance="raised"
      color="primary">
      Seleccionar vista 0
    </mv-button>

    <mv-button
      (clickEvent)="selectItem(1)"
      appearance="raised"
      color="primary">
      Seleccionar vista 1
    </mv-button>

    <mv-button
      (clickEvent)="selectItem(2)"

Menú en el último botón.
code
La propiedad showMenu permite definir un menú al final de los botones principales del menú. Aparte de poner la variable showMenu a true también debemos definir el menú en sí como se muestra a continuación:

tabItems
Definición de los botones.
code
La propiedad tabItems permite definir los distintos botones de los que se compone el menú. Como máximo se podrán definir 6 elementos más el menú.

Hay que definir por un lado un array con tantas posiciones como botones queramos que tenga el componente. En cada posición del array debemos definir el icono, la familia de iconos y el título del botón:

icon: nombre del icono (de Material o Font Awesome)
fontSet: 'fa' (si queremos que sea Font Awesome. Material por defecto)
title: título del botón
Por otro lado hay que definir el html del contenido que queremos visualizar al pulsar cada botón con las etiquetas 'content1', 'content2'...

HTML
TS

  <mv-footer-tab
    color="corporative"
    [tabItems]="arrayTabItems2">

    <div content1>
      Contenido1
    </div>
    <div content2>
      Contenido2
    </div>
    <div content3>
      Contenido3
    </div>
    <div content4>
      Contenido3
    </div>
    <div content5>
      Contenido3
    </div>
  </mv-footer-tab>
