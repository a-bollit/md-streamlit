Información del componente mv-footer-nav-tab.
code

navBarConfig
Definición de los botones y menú.
code
La propiedad navBarConfig permite definir los distintos botones de los que se compone la barra de navegación.

Hay que definir por un lado un array con tantas posiciones como botones queramos que tenga el componente. En cada posición del array debemos definir el icono, el título y la url de navegación:

icon: nombre del icono
fontSet: 'fa' (si queremos que sea Font Awesome. Material por defecto)
title: título que se mostrará debajo del icono
stateUrl: url de navegación
También se puede definir un botón más a modo de menú, que desplegará un menú contextual con tantas opciones como definamos:

title: título que se mostrará debajo del icono
stateUrl: url de navegación
