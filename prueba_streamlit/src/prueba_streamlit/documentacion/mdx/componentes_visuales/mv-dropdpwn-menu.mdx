Información
Información del componente mv-dropdown-menu.
code
El componente mv-dropdown se basa en el componente mat-menu de Angular Material, al que se le añade funcionalidad y estilos adicionales.

Sirve para desplegar un menú contextual en cualquier parte de la pantalla.

Ejemplo de uso
Uso básico del componente mv-dropdown-menu.
code
A continuación se muestra un ejemplo básico de uso del componente.

HTML
TS

  <mv-button
    color="primary"
    appearance="raised"
    [matMenuTriggerFor]="menu">
    Lanzar menú
  </mv-button>

  <mv-menu #menu="mvMenu">
    <button mv-menu-item>Item 1</button>
    <button mv-menu-item>Item 2</button>
  </mv-menu>

xPosition
Posición horizontal del menú.
code
La propiedad xPosition permite definir la posición horizontal donde aparecerá el menú respecto al elemento que lo lanza.

HTML
TS

    color="primary"
    appearance="raised"

    Lanzar menú (before)
  </mv-button>

    <button mv-menu-item>Item 2</button>
  </mv-menu>

yPosition:

openMenu
Abrir el menú desde el código TS.
code
La función openMenu permite abrir el menú desde el código TS.

Aunque se abra el menú por código, hay que definir un elemento lanzador "mvMenuTrigger"

HTML
TS

  <mv-button
    color="primary"
    appearance="raised"
    (clickEvent)="openMenuFunctionExample()">
    Abrir menú
  </mv-button>

  <div 
    #menuTriggerOpen="mvMenuTrigger"
    [mvMenuTriggerFor]="openMenuExample">
  </div>
  <mv-menu #openMenuExample="mvMenu">
    <button mv-menu-item>Item 1</button>
    <button mv-menu-item>Item 2</button>
  </mv-menu>
  
openMenu:

closeMenu
Cerrar el menú desde el código TS.
code
La función closeMenu permite cerrar el menú desde el código TS.

Aunque se cierre el menú por código, hay que definir un elemento lanzador "mvMenuTrigger"

HTML
TS

  <mv-button
    color="primary"
    appearance="raised"
    (clickEvent)="closeMenuFunctionExample()">
    Abrir menú
  </mv-button>

  <div
    #menuTriggerClose="mvMenuTrigger"
    [mvMenuTriggerFor]="closeMenuExample">
  </div>
  <mv-menu #closeMenuExample="mvMenu">
    <button mv-menu-item>Item 1</button>
    <button mv-menu-item>Item 2</button>
  </mv-menu>
