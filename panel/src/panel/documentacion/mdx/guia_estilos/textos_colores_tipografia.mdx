Textos y tipografía general
code
La tipografía principal de MOVA es "Montserrat".

IMPORTANTE
Si se quiere usar texto en negrita hay que hacerlo usando la etiqueta <strong>, ya que es la adecuada por cuestiones de accesibilidad. En la medida de los posible no usar la etiqueta <b> de html.

Así mismo, no usar la etiqueta <i> de html para mostrar texto en cursiva o destacado. Usar en su defecto la etiqueta <em> por el mismo motivo.

En la medida de lo posible es recomendable usar las variables que proporciona MOVA porque está todo desarrollado en base a muchas de estas variables y si usamos otras puede ocasionar comportamientos o aspectos indeseados. Por ejemplo, en caso de necesitar media-queries, se recomienda hacerlo con las variables que MOVA proporciona para ello, porque los componentes están desarrollados en base a esas variables. Otro ejemplo puede ser los z-index, si usamos un z-index con valor al azar podría ocasionar conflictos con otros componentes.

No obstante, se pueden crear todas las variables que sean necesarias para aspectos concretos de nuestra app que MOVA no proporcione. En caso de crearlas es recomendable situarlas en el archivo styles.scss por mantenerlas centralizadas y para un mejor man
