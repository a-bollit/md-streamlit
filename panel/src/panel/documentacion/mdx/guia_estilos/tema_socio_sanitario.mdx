En MOVA se puede elegir el tema visual socio sanitario, que se corresponde con los elementos en azul en vez del rojo corporativo. Para configurarlo:

Escoger 1 (rojo) o 2 (azul) en la variable style del archivo config-default-main-frame.json

Escoger uno u otro tema (eliminando o comentando el que no queramos) en la variable CSS del archivo styles.scss

config-default-main-frame.json
styles.scss

  {
    "generalConfig": {
      "style": 2
    }
    ...
  }
