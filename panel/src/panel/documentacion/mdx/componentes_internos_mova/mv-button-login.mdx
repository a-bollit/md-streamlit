Información
Información del componente mv-button-login.
code
El componente mv-button-login sirve para disponer de un botón con funcionalidad directa para logarse y deslogarse en la app. Si estamos deslogados y pulsamos en el botón, nos logaremos y el aspecto del botón cambiará, de manera que si volvemos a pulsar nos deslogaremos.

loginFontSet
Familia de iconos para login.
code
La propiedad loginFontSet permite definir la familia de iconos que queremos usar cuando la acción a realizar es el login, tan solo en caso de querer usar Font Awesome. Si queremos usar los iconos de Material no hay que usar esta propiedad.

HTML
TS

  <mv-button-login
    loginFontSet="far"
    loginIcon="fa-address-book"
    showIcon=true>
  </mv-button-login>

  <mv-button-login
    loginFontSet="fas"
    loginIcon="fa-address-book"
    showIcon=true>
  </mv-button-login>

  <mv-button-login
    loginIcon="build"
    showIcon=true>
  </mv-button-login>
