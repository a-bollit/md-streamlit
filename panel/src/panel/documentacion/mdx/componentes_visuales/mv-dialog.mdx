Información
Información del componente mv-dialog.
code
El componente mv-dialog se basa en el componente mat-dialog de Angular Material, al que se le añade funcionalidad y estilos adicionales.

Sirve para mostrar un diálogo que se adapta al dispositivo, con la información deseada.

Ejemplo de uso
Uso básico del componente mv-dialog.
code
A continuación se muestra un ejemplo básico de uso del componente.

TS

    appearance="raised"

successCallback
Función de callback.
code
La propiedad successCallback permite definir la acción a realizar cuando pulsamos algún botón del diálogo.

La función de callback devolverá el resultado de la acción.

Devuelve un objeto con los siguientes parametros, marcando a true el botón que el usuario hubiera marcado

accept: Al hacer click al botón de aceptar
cancel: Al hacer click al botón de cancelar
out: Al hacer click fuera del dialogo
close: Al cerrar el dialogo
HTML
TS
RETURN

  <mv-button
    color="primary"
    appearance="raised"
    (clickEvent)="exampleSuccessCallbackExample()">
    Lanzar diálogo
  </mv-button>
