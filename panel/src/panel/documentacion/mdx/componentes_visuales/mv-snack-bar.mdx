Información
Información del componente mv-snack-bar.
code
Este componente sirve para mostrar un mensaje breve de información, aviso o error en la pantalla durante unos segundos.

Ejemplo de uso
Uso básico del componente mv-snack-bar.
code
A continuación se muestra un ejemplo básico de uso del componente.

HTML
TS

  <mv-button
    color="primary"
    appearance="raised"
    (clickEvent)="exampleSnackBar()">
    Lanzar snack bar
  </mv-button>
  
Ejemplo de snack bar:

actionText
Texto del botón.
code
La propiedad actionText permite definir el texto del botón del componente.

Si pulsamos el botón, el snack bar se ocultará

HTML
TS

  <mv-button
    color="primary"
    appearance="raised"
    (clickEvent)="exampleSnackActionTextExample()">
    Lanzar snack bar
  </mv-button>

type
Color del snack bar.
code
La propiedad type permite definir el color del componente entre info, warning y danger.

El color por defecto es warning.
