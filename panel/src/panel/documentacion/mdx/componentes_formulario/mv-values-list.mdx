Información
Información del componente mv-values-list.
code
Este componente sirve para mostrar u obtener una gran cantidad de información de la manera más sencilla posible. Se pueden cargar datos del servidor o locales. Al abrir el diálogo podremos filtrar entre todos los elementos, paginar, ordenar...

Ejemplo de uso
Uso básico del componente mv-values-list.
code
A continuación se muestra un ejemplo básico de uso del componente con origen de datos en servidor y en local.

La carga de los datos se hace mediante el uso del componente mv-table-responsive. Consulta la documentación para más información.

dataSourceLoader
Datos cargados desde servidor.
code
La propiedad dataSourceLoader permite cargar los datos desde el servidor. Además de la propiedad, hay que definir lo siguiente:

Un servicio rest que obtenga los datos deseados

Una clase que implemente mvLoadData, que servirá de interfaz entre el servicio y el values-list

En el código asociado al values-list, se debe crear un objeto de la clase LoadData, al que se le debe pasar el servicio que obtiene los datos

Si se quiere definir un paginador no se hará con la propiedad pageSize, sino pasando el parámetro al servicio que obtiene los datos

delay
Retardo antes de realizar la búsqueda.
code
La propiedad delay crea un retardo de los milisegundos especificados, antes de realizar la búsqueda.

validateFirst
Validación inicial del componente.
