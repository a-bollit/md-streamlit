Información del componente mv-textarea.
code
El componente mv-textarea se basa en el componente mat-textarea de Angular Material, al que se le añade funcionalidad y estilos adicionales.

keyupEvent
Evento keyup.
code
El evento keyupEvent permite definir una función que se ejecutará cuando el usuario deja de teclear

La diferencia con el evento changeEvent es que no se ejecutará la función keyupEvent hasta que el usuario deje de teclear, de manera que si mantiene pulsado de manera continuada un valor del teclado se estará introduciendo muchos caracteres sin que se ejecute la función keyupEvent, sin embargo sí se estaría ejecutando la función changeEvent porque el valor sí está cambiando.

HTML
TS

  <mv-textarea
    (keyupEvent)="keyupEventFunction($event)">
  </mv-textarea>
