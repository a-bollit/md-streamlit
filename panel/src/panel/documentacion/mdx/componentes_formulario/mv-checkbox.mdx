Información
Información del componente mv-checkbox.
code
El componente mv-checkbox se basa en el componente mat-checkbox de Angular Material, al que se le añade funcionalidad y estilos adicionales.

indeterminate
Estado indeterminado.
code
La propiedad indeterminate permite definir un estado visual indeterminado, para cuando queremos evidenciar que el componente no está ni a true ni a false.

Es importante señalar que, aunque el componente esté en un momento dado con indeterminate = true, el valor del model no cambia, será el que tuviera el componente antes de ponerse indeterminado

HTML
TS

  <mv-checkbox-x
    [(ngModel)]="checkbox006Model"
    [(indeterminate)]=setIndeterminateModel>
  </mv-checkbox-x>

  <mv-button
    (clickEvent)="setIndeterminate()"
    color="primary">
    Poner indeterminate a true
  </mv-button>

indeterminateChangeEvent
Validación del componente en cada cambio.
code
La propiedad indeterminateChangeEvent permite definir una función que se ejecutará cuando el valor del componente pase de indeterminado a true o false y viceversa, cuando pase de true o false a indeterminado

Este evento solo tiene sentido cuando el componente tiene definida la propiedad indeterminate = true

TS

    [(ngModel)]="checkbox017Model"
    [(indeterminate)]=setIndeterminateModel2
    (indeterminateChangeEvent)=indeterminateChangeFunction()>
  </mv-checkbox-x>

  <mv-button
    (clickEvent)="setIndeterminate2()"
    color="primary">
    Poner indeterminate a true

doValidate
Validación del componente.
code
La propiedad doValidate permite validar el componente para que se muestren los posibles mensajes de error. Se debe usar este evento, por ejemplo, cuando queremos validar el componente en un formulario al hacer clic en el botón de guardar/submit.

HTML
TS

  <mv-checkbox-x
    #mvCheckboxValidateEx
    required=true
    validateOnType=false
    validateFirst=false>
  </mv-checkbox-x>

  <mv-button
    (clickEvent)="mvCheckboxValidateEx.doValidate()"
    color="primary">
    Validar
  </mv-button>
  

ariaLabel
Etiqueta para accesibilidad
code
La etiqueta ariaLabel se utiliza para proporcionar una descripción de texto alternativa para elementos que no tienen un texto visible por si mismo y aseguarse que usuarios con lectores de pantalla y otros asistentes de accesibilidad puedan entender el propósito del componente.

HTML
TS

  <mv-checkbox-x
    arialLabel="Tu texto personalizado">
  </mv-checkbox-x>
