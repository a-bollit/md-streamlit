Consideraciones generales sobre Imágenes WMTS

Podemos generar capas de imágenes WMTS a través de servicios WMTS. Para ello, debemos establecer typeLayer como 'Image' y subtypeLayer como 'WMTS' en la contrucción de un objeto de tipo apigis.Layer() como se explica en la documentación general sobre Servicios.

IMPORTANTE

Para poder visualizar una capa, deberá estar contenida dentro de un objeto de tipo apigis.Map.

Así mismo, un objeto de tipo apigis.Map deberá estar contenido dentro de un objeto de tipo apigis.Viewer.

Métodos Métodos comunes a todas las capas cuyo mapType: 'OL', typeLayer: 'Image' y subtypeLayer: 'WMTS'.

Se muestran a continuación los métodos comunes a todas los objeto de tipo apigis.Layer cuyo mapType: 'OL', typeLayer: 'Image' y subtypeLayer: 'WMTS':


initialized

Permite saber si el servicio está inicializado (el servicio se inicializa una vez se han recuperado los capabilities del mismo).


setCapabilities(value)

Configura el nombre del servicio. Sus argumentos son:

  • value <string > Capabilities del servicio.

moveLayer(indexTo)

Cambia la posición de la capa dentro del mapa. Sus argumentos son:

  • indexTo <number > Nuevo índice en el mapa.

refresh()

Refresca la capa.
