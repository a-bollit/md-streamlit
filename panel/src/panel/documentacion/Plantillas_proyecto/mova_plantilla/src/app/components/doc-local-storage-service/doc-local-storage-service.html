Información Información del servicio local-storage-service.

Este servicio sirve de interfaz para acceder al localStorage del navegador. Se puede usar en web y móvil.
code
init Resetear el nodo principal.

El evento init permite resetear el nodo principal de localStorage donde se almacena la información de MOVA.

init(reset)

reset

  • true: resetea el nodo principal de localStorage
  • false: resetea el nodo principal de localStorage si está creado y lo crea en caso contrario

Resetear el nodo prinicpal de mova mediante el evento init:

init(true)

Recuperar el nodo de MOVA en localStorage:

getItems()

Nodo principal de MOVA en localStorage:

{{movaLocalStorageNode | json}}
code
setItem Almacenar información en localStorage.

El evento setItem permite almacenar información en el nodo principal de localStorage donde se almacena la información de MOVA.

setItem(key, value, crypt)

key

  • 'string': cadena con la clave del objeto que estamos almacenando

value

  • 'string': valor que estamos almacenando

crypt

  • true: encripta el valor que estamos almacenando
  • false: no encripta el valor que estamos almacenando

Almacenar en el nodo de MOVA en localStorage el texto de 'lorem ipsum' sin encriptar:

setItemExample('DocLocalStorageServiceExample', 'lorem ipsum...', false)

Almacenar en el nodo de MOVA en localStorage el texto de 'lorem ipsum' encriptado:

setItemExample('DocLocalStorageServiceExample', 'lorem ipsum...', true)

Recuperar el nodo de MOVA en localStorage:

getItems()

Nodo principal de MOVA en localStorage:

{{movaLocalStorageNode | json}}
code
getItem Recuperar un objeto concreto de localStorage.

El evento getItem permite recuperar un objeto concreto del nodo principal de localStorage donde se almacena la información de MOVA.

getItem(key)

key

  • 'string': cadena con la clave del objeto que queremos recuperar

Almacenamos con clave 'DocLocalStorageServiceExample' en el nodo de MOVA en localStorage:

setItemExample('DocLocalStorageServiceExample', 'lorem ipsum...', false)

Recuperar el objeto de clave 'DocLocalStorageServiceExample' del nodo de MOVA en localStorage:

getItemExample('DocLocalStorageServiceExample')

Objeto de clave 'DocLocalStorageServiceExample' del nodo principal de MOVA en localStorage:

{{movaLocalStorageNodeKey | json}}
