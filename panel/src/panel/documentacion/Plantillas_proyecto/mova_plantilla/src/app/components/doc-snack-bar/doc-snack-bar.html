Información Información del componente mv-snack-bar.

Este componente sirve para mostrar un mensaje breve de información, aviso o error en la pantalla durante unos segundos.

actionText Texto del botón.

La propiedad actionText permite definir el texto del botón del componente.

Si pulsamos el botón, el snack bar se ocultará

Con actionText = 'Deshacer':

Lanzar snack bar

type Color del snack bar.

La propiedad type permite definir el color del componente entre info, warning y danger.

El color por defecto es warning.

verticalPosition Posición vertical del snack bar.

La propiedad verticalPosition permite definir la posición vertical donde se mostrará el componente. Los valores aceptados son:

  • top
  • bottom

El valor por defecto es 'bottom'.

Con verticalPosition = 'top':

Lanzar snack 'top'

Con verticalPosition = 'bottom':

Lanzar snack 'bottom'
