indeterminate Estado indeterminado.

La propiedad indeterminate permite definir un estado visual indeterminado, para cuando queremos evidenciar que el componente no está ni a true ni a false.

Es importante señalar que, aunque el componente esté en un momento dado con indeterminate = true, el valor del model no cambia, será el que tuviera el componente antes de ponerse indeterminado

Indeterminate = 'true'

doValidate Validación del componente.

La propiedad doValidate permite validar el componente para que se muestren los posibles mensajes de error. Se debe usar este evento, por ejemplo, cuando queremos validar el componente en un formulario al hacer clic en el botón de guardar/submit.

ariaLabel Etiqueta para accesibilidad

La etiqueta ariaLabel se utiliza para proporcionar una descripción de texto alternativa para elementos que no tienen un texto visible por si mismo y aseguarse que usuarios con lectores de pantalla y otros asistentes de accesibilidad puedan entender el propósito del componente.
