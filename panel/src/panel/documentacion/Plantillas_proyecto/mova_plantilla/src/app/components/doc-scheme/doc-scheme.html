Información Información del scheme.

El custom scheme se usa para identificar la app en el caso de salir de la misma y luego tener que volver. Por ejemplo al salir a la web de autologin.

Configuración Configuración.

El esquema de la app se configura en el archivo config.xml y deberá ser único para cada app.

Importante

El esquema no debe incluir mayúsculas para un correcto funcionamiento.
