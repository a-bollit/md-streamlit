Información Información de la directiva mv-show-when.

Esta directiva se puede usar en cualquier componente para mostrarlo u ocultarlo según estemos autenticados o no, de manera que un mismo componente se muestre o se oculte dependiendo de si estamos o no logados

mvShowWhen Cuándo mostrar componente.

La propiedad mvShowWhen permite definir si queremos que se muestre un componente cuando estamos logados o queremos que se muestre cuando no estamos logados.
