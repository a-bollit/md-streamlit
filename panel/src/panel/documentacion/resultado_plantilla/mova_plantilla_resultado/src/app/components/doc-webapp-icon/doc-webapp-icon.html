Información Información.

Las aplicaciones PWA (Progressive Web App) permiten que, al entrar con el navegador del móvil a una web, podamos añadir dicha web a la pantalla de inicio de nuestro dispositivo con su icono correspondiente, como si fuera una app a modo de atajo para entrar directamente al sitio web.

En los dispositivos iOS tendremos que añadir manualmente la web a la pantalla de inicio, mientras que en los dispositivos Android nos aparecerá un mensaje para añadirlo directamente pulsando en el mensaje.

Configuración Configuración para establecer app como icono

Para configurar esta funcionalidad debemos definir los siguientes elementos:

  • Modificar el archivo llamado manifest.webmanifest con los valores propios de nuestra aplicación
  • Crear las imágenes e iconos necesarios referenciados en el manifest
  • Modificar el index.html de nuestro proyecto
  • Habilitar el service worker en app.module.ts
