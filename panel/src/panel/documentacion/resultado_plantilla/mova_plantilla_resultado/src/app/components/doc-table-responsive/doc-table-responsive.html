Información Información del componente mv-table-responsive.

El componente mv-table-responsive se basa en el componente mat-table de Angular Material, al que se le añade funcionalidad y estilos adicionales.

data Datos cargados desde cliente.

La propiedad data permite definir los datos del componente a partir de un array en el cliente.
code
dataSourceLoader Datos cargados desde servidor.

La propiedad dataSourceLoader permite cargar los datos desde el servidor. Además de la propiedad, hay que definir lo siguiente:

  • Un servicio rest que obtenga los datos deseados

  • Una clase que implemente mvLoadData, que servirá de interfaz entre el servicio y el values-list

  • En el código asociado al values-list, se debe crear un objeto de la clase LoadData, al que se le debe pasar el servicio que obtiene los datos

  • Si se quiere definir un paginador no se hará con la propiedad pageSize, sino pasando el parámetro al servicio que obtiene los datos

formatType Formato visual.

La propiedad formatType permite especificar el formato visual de los datos.

  • card: Componente con formato de tarjeta.
  • card-responsive: Componente con formato de tarjeta que se adapta de manera responsiva.
  • card-inline: Tarjeta en línea, colocada horizontalmente.
  • card-inline-responsive: Tarjeta en línea, colocada horizontalmente, con diseño responsivo.
  • card-inline-truncate: Tarjeta en línea con texto truncado para que ocupe un espacio limitado.
  • card-inline-truncate-responsive: Tarjeta en línea con texto truncado para diseño responsivo.
