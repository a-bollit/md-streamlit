Descripción

Esto es un ejemplo del componente de mova mvFooterMenu que usamos para hacer un menú en el pie. Al hacer click en cada pestaña o tab se navega a otro contenido definido por el desarrollador.

Este comportamiento de navegación entre páginas se podría usar para sustituir el menú principal, o para los accesos directos a las opciones más usadas siempre disponibles en la pantalla, por ejemplo.
