Información

Las vistas de ejemplo que se pueden ver pulsando en el botón Acceder son un listado de alumnos, con posibilidad de creación, edición y borrado.

Es necesario estar logado con WSO2 para poder acceder. Las credenciales son:

  • Usuario: movausu2
  • Contraseña: movausu2

Las llamadas a los distintos recursos REST están securizadas con el envío de un token de acceso. Este token se obtiene con el login de WSO2 y su manejo es transparente para el desarrollador.

Para más información de WSO2 ver la documentación: doc-login-view

Algunos de los datos que se muestran son aleatorios, no almacenándose en base de datos:

  • Imágenes de los alumnos
  • Notas de las asignaturas
  • Observaciones de las asignaturas
  • Provincia
  • Municipio
  • Archivo PDF
  • Sexo
  • Solicitante de beca
