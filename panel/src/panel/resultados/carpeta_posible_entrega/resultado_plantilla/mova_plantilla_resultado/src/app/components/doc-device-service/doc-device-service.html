Información Información del servicio device-service.

Este servicio proporciona funcionalidad relacionada con las propiedades del dispositivo en el que se ejecuta la app.
code
getMobileInfo Información del dispositivo

Devuelve información del dispositivo movil.

Valores que devuelve:

  1. cordovaVersion: Versión de cordova

  2. isSimulator: Si está siendo ejecutado en un simulador

  3. model: Modelo del dispositivo

  4. manufacturer: Compañía que manufacturó el dispositivo

  5. serialNumber: Nº de serie del dispositivo

  6. uuid: UUID del dispositivo

  7. platform: Sistema operativo del dispositivo

  8. version: Versión del sistema operativo del dispositivo

Ejemplo:

loadGetMobileInfo 
 {{mobileInfoResponse | json}}
code
getWebInfo Información del navegador.

Devuelve la información del navegador

Valores que devuelve:

  1. browserCodeName: Contiene el nombre código del navegador

  2. browserEngine: Contiene el número de compilación del navegador

  3. browserName: Contiene un DOMString con el nombre oficial del navegador

  4. browserPlatform: Contiene el nombre de la plataforma del navegador

  5. browserVersionInfo: Contiene la versión del navegador

  6. browserUserAgent: Contiene la cadena del userAgent

  7. isCookieEnabled: Indica si el navegador tiene activadas las cookies

  8. language: Contiene una cadena representando el idioma del navegador

  9. platform: Plataforma del navegador

Ejemplo:

loadGetWebInfo 
 {{webInfoResponse | json}}
