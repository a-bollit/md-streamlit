Información Información del archivo config-default-main-frame.json.

Este archivo permite definir el aspecto visual general de la app.

Para configurar los aspectos generales de la app ver la documentación de config-default.json

Para configurar aspectos según el entorno (desarrollo/validación/producción) ver la documentación de environment.ts, environment.val.ts y environment.prod.ts

Configuración Configuración del archivo config-default-main-frame.json.
  • generalConfig
    • style - Variable para definir el tema principal de la app: 1 rojo corporativo o 2 azul socio sanitario. Este cambio se aplicará al color del header y a algunos elementos visuales de los componentes MOVA.
    • showLoadingMode - Variable para definir el icono de loading: "mova1" para icono corporativo similar a mova1 o "mova2" para spinner.
  • headerConfig
    • fixed - Variable para establecer el header fijo al hacer scroll.
    • showBackButton - Variable para definir si queremos mostrar el botón de atrás en el header.
    • showHeader - Variable para definir si queremos mostrar u ocultar el header.
    • showHeaderButtonLogin - Variable para definir si queremos mostrar el botón de login en el header.
    • showLateralMenu - Variable para definir si queremos que exista el menú lateral.
    • showMenuButton - Variable para definir si queremos mostrar el botón de abrir/cerrar el menú en el header.
    • type - Variable para definir el tema del header: 'white-corporative' (color base blanco y el complementario será rojo o azul según lo que hayamos configurado en la variable style) o '' (color base será rojo o azul y el color complementario será el blanco).
    • headerButtonLoginFontSet - Variable para definir la familia de iconos del botón de login del header (doc-button-login).
    • headerButtonLogoutFontSet - Variable para definir la familia de iconos del botón de login del header cuando estamos logados (doc-button-login).
    • headerButtonLoginIcon - Variable para definir el icono del botón de login del header (doc-button-login).
    • headerButtonLogoutIcon - Variable para definir el icono del botón de login del header cuando estamos logados (doc-button-login).
  • lateralMenuConfig
    • appShortName - Variable para definir un nombre corto que aparecerá en el menú lateral.
    • appLongName - Variable para definir un nombre largo que aparecerá en el menú lateral, debajo de appShortName.
    • appDesc - Variable para definir un texto que aparecerá en el menú lateral, debajo de appLongName.
    • position - Variable para definir la posición del menú lateral: 'start' (a la izquierda de la pantalla) o 'end' (a la derecha de la pantalla).
    • mode - Variable para definir el comportamiento del menú lateral al abrirse en pantallas grandes: 'over' (por encima del contenedor principal), 'push' (empuja al contenedor principal) o 'side' (permanece fijo junto al contenedor principal).
    • modeSmall - Variable para definir el comportamiento del menú lateral al abrirse en pantallas pequeñas: 'over' (por encima del contenedor principal), 'push' (empuja al contenedor principal) o 'side' (permanece fijo junto al contenedor principal).
    • initOpen - Variable para definir el comportamiento del menú lateral al iniciar la app: true (el menú estará abierto en pantallas grandes), false (el menú permanecerá cerrado de primeras).
    • alwaysOpen - Variable para definir si queremos que el menú lateral permanezca abierto por defecto.
    • logoApp - Variable para definir la ruta de la imagen que se usará como logo de la app en el menú lateral.
    • closeOnSelectItem - Variable para definir si queremos que el menú lateral se cierre al hacer click fuera del mismo.
    • showLateralMenuButtonLogin - Variable para definir si queremos mostrar el botón de login en el menú lateral.
    • showLateralMenuButtonLoginIfLogged - Variable para definir si queremos mostrar el botón de login en el menú lateral cuando estamos logados.
    • lateralMenuButtonLoginAppearance - Variable para definir la apariencia del botón de login del menú lateral (doc-button-login).
    • lateralMenuButtonLoginColor - Variable para definir el color del botón de login del menú lateral (doc-button-login).
    • lateralMenuButtonLoginColorLogout - Variable para definir el color cuando estamos logados del botón de login del menú lateral (doc-button-login).
  • navBarConfig
    • showNavBar - Variable para definir si queremos visualizar el componente doc-footer-nav-tab.
    • navBarColor - Variable para definir el color del componente.
    • navBarColorBar - Variable para definir el color de la barra de navegación.
    • navBarLinks - Variable para definir el array de botones. Cada elemento del array estará compuesto de:
      • title - Variable para definir el título del botón.
      • stateUrl - Variable para definir la url del estado al que se navega al pulsar.
      • fontSet - Variable para definir la fuente del icono.
      • icon - Variable para definir el icono del botón.
    • navBarMenu - Variable para definir el array de botones. Cada elemento del array estará compuesto de:
      • title - Variable para definir el título del botón.
      • stateUrl - Variable para definir la url del estado al que se navega al pulsar.
