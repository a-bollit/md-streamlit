Información Información sobre compilación.

Los distintos modos de compilación de una app MOVA son:

  • npm run serve-des - compilación en local con entorno de desarrollo. Sólo para ejecución en el navegador con localhost
  • npm run serve-val - compilación en local con entorno de validación. Sólo para ejecución en el navegador con localhost
  • npm run serve-pro - compilación en local con entorno de producción. Sólo para ejecución en el navegador con localhost
  • npm run build-debug-des - compilación de archivos sin optimizar (mejor para depurar) con entorno de desarrollo.
  • npm run build-debug-val - compilación de archivos sin optimizar (mejor para depurar) con entorno de validación.
  • npm run build-debug-pro - compilación de archivos sin optimizar (mejor para depurar) con entorno de producción.
  • npm run build-release-des - compilación de archivos optimizados con entorno de desarrollo.
  • npm run build-release-val - compilación de archivos optimizados con entorno de validación.
  • npm run build-release-pro - compilación de archivos optimizados con entorno de producción.

Los primeros tres modos se usarán para ejecutar la app en local en el navegador, pero no se generará la app compilada
