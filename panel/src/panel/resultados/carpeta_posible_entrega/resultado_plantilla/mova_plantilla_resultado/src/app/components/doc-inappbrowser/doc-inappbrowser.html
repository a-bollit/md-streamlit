Información Información de inAppBrowser.

El plugin inAppBrowser sirve para abrir una vista del navegador del dispositivo con la url deseada.

Documentación de inAppBrowser (https://github.com/apache/cordova-plugin-inappbrowser)

El servicio diferencia el dispositivo desde el que se ejecuta y lo configura. Para dispositivos móviles ejecuta el plugin de cordova inAppBrowser y para web usa window.open()

Posibles fallos Errores conocidos de inAppBrowser

En algunos casos se ha dado que inAppBrowser no funciona correctamente en versiones de Android api 30 o superiores debido a que el plugin de safariViewController tiene un problema en una de sus versiones que le falta por añadir un código para su correcto funcionamiento y hay que añadirlo a mano en el config.xml del proyecto en caso de que no esté.
