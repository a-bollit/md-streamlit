Documentación sobre Impresión

El API GIS permite imprimir los mapas que se construyen de dos formas: impresión en servidor y impresión en cliente. La clase apigis.Map permiten la herencia de servidores para aumentar la funcionalidad de los elementos.

En esta página se muestra la forma en la que se puede realizar una impresión en cliente o en servidor.

  1. linkImpresión en servidor
  2. linkImpresión en cliente
