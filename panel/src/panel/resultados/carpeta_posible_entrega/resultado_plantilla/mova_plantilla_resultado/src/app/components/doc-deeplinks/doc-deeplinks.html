¿Que es un deeplink? Deeplink

Un deepLink es un tipo de enlace <a href=""> que permite enlazar a una app en vez de a una página web normal.

Al igual que a una página se le pueden pasar una pagina o vista destino y unos parámetros

Esta funcionalidad se está usando actualmente en MOVA para cuando se usa el broker de identidades y se sale de la app para ir al broker de identidades luego volver a la app mediante un enlace.
