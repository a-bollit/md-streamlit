Integración del API GIS en MOVA2 Información sobre la integración del API GIS en MOVA2.

El API GIS se compone de una serie de puntos de entrada que los integradores deben consumir para desarrollar sus propios visores, mapas, servicios... etc.

Para tener acceso a todos los puntos de entrada del API GIS, el integrador deberá instalar el API GIS situándose en el directorio del proyecto MOVA2 y con la consola ejecutar "npm install http://e_jpb71@gitlab.madrid.org/GIS_open/GIS_DIFUSION.git#develop".

IMPORTANTE

Para que el API GIS funcione correctamente, es necesario que el integrador instale también las dependencias que necesite. Puede obtenerse más información en Dependencias del API GIS.
