Información Información del componente mv-header.

El componente mv-header se basa en el componente mat-toolbar de Angular Material, al que se le añade funcionalidad y estilos adicionales.

Sirve para mostrar y configurar la cabecara de la app.
code
Configuración Configuración del componente mv-header.

El componente mv-header de una aplicación MOVA es configurable mediante el archivo config-default-main-container.json:, concretamente dentro del objeto headerConfig

Aparte de la configuración inicial del archivo config-default-main-container.json, se pueden definir otros valores para la cabecera en cada componente que sobrescribirá la configuración inicial, como se ve a continuación

fixed Cabecera fija.

La propiedad fixed permite definir la cabecera fija:

  • true: la cabecera permanecerá fija y visible independientemente del scroll de la página
  • false: la cabecera desaparecerá en cuanto se haga scroll hacia abajo
Cambiar fixed
code
showBackButton Mostrar botón de atrás.

La propiedad showBackButton permite mostrar u ocultar el botón de navegación hacia atrás:

  • true: se mostrará el botón
  • false: se ocultará el botón
Cambiar showBackButton
code
showMenuButton Mostrar botón del menú.

La propiedad showMenuButton permite mostrar u ocultar el botón de abrir/cerrar el menú:

  • true: se mostrará el botón
  • false: se ocultará el botón
Cambiar showMenuButton
code
type Tema de la cabecera.

La propiedad type permite definir el tema visual de la cabecera:

  • 'white-corporative': la cabecera será blanca y el resto de elementos serán de color corporativo

El caso por defecto será con la cabecera de color corporativo y el resto de elementos serán de color blanco
Cambiar type
