Ejemplo de tablas

El framework incorpora un componente para mostrar tablas con distintas configuraciones y formas de visualización.

Para ver las opciones con todo detalle, dirigirse a la documentación del componente: doc-table-responsive

Ejemplo sin formato:

Se mostrará siempre en modo tabla, y si la pantalla es más pequeña que la tabla deberemos hacer scroll horizontal para visualizar todos las columnas.

Ejemplo con formato 'card-responsive':

Se mostrará en modo tabla excepto si el tamaño de la pantalla es pequeño, que pasará a mostrarse en modo card, que consiste en tantas 'tarjetas' como filas de datos haya:

Ejemplo con formato 'card-inline-responsive':

Se mostrará en modo tabla, y si la pantalla es más pequeña que la tabla deberemos se mostrará en modo card-inline, que es en modo 'tarjeta' con una sola línea para el título y el dato.

Ejemplo con formato 'card-inline-truncate-responsive':

Se mostrará en modo tabla, y si la pantalla es más pequeña que la tabla deberemos se mostrará en modo card-inline-truncate, que es en modo 'tarjeta' con una sola línea para el título y el dato, truncando el valor si supera el ancho del contenedor
