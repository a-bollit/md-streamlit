Información Información de generación.

Para la generación automática de iconos y splashscreens hay herramientas por internet que ayudan en este paso, nosotros vamos a documentar el uso de abiro

https://pgicons.abiro.com/

una herramienta especifica para la generación de imagenes para cordova
code
Configuración Configuración de la herramienta.

El interface de la herramienta es simple, viene con una pequeña configuración y unos botones para subir nuestras imágenes y generar los iconos y splashscreen

tiene dos botones para seleccionar las imagenes de la app, una para el icono y otra opcional para el fondo del splashscreen. En caso de no seleccionar una imagen para el fondo, nos deja elegir el color en: #rrggbbaa/#rgb, rgba()/hsb()/hsl() y el nombre del color

También tiene opciones para seleccionar el tipo de dispositivo (Android, iOS(Legacy y storyboard), windows) que vienen por defecto en portrait, pero también se puede especificar que vengan en landscape.

Tiene también unas configuraciones para ajustar la imagen en el icono y el splashscreen

Información Información de la imagen.

Subir un icono de la aplicación preferiblemente cuadrado y un fondo de pantalla para la splash screen opcional en un formato de imagen de alta resolución, pero preferiblemente PNG, cada uno de ellos de menos de 16 megapíxeles. Ambas imágenes pueden tener transparencia/alfa.

Recomendaciones:

  • Lea sobre iconos y splash screens de cordova
    • Iconos custom - Apache Cordova
    • Splashscreen - Apache Cordova
  • Las imágenes deben tener al menos 1 megapixel para que tenga la calidad suficiente, pero no mas de 16 megapixeles
  • Evitar fotografias o imagenes con gradientes si es posible
  • En caso de timeout en la herramienta, prueba a convertir las imagenes por plataforma

Ejemplo Ejemplo de uso

Para la generación de ejemplo voy a usar un icono de la aplicación de mova

Configuro la herramienta seleccionando mi imagen de ejemplo, un color para el background de fondo, dejo el resto de la configuración por defecto.

Una vez configurado, le doy a Generate Images para que genere todas las imagenes del proyecto, esto puede llevarle un momento.

Nos aparecera un botón nuevo para iniciar la descarga del zip con todas las imágenes.
