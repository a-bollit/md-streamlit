Constructor apigis.Loader

Se puede generar directamente uno o varios visores junto con mapas, capas y simboogía mediante la clase Constructor apigis.Loader.

new ApiLoader(config, headerSecurty = false, config_functions = {}, custom_options = [], create_viewers = true)

  • config <object > Configuracion para generar la visualización de Viewer y Mapa. Puede contener las propiedades:
    • viewers <object > Objeto contiene la configuración para los visores.
    • maps <object > Objeto contiene la configuración para los mapas.
    • layers <object > Objeto contiene la configuración para las capas.
    • symbols <object > Objeto contiene la configuración para la simbología.
  • headerSecurty = false <boolean > Cabecera de seguridad si es necesaria para los SLDs.
  • config_functions <object > Objeto con las funciones a aplicar.
  • custom_options <[] > Custom options a anadir a las props de cada objeto.
  • create_viewers <boolean > Indica si el constructor tiene que crear los viewer.

Se puede consultar información más detallada y ejemplos sobre el uso de apigis.Loader en el documento de Arquitectura sobre el API GIS.
