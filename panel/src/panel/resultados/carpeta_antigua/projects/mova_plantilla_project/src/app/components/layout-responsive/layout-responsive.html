  • Fichas

A continuación se muestra la implementación de una botonera de la que se desconoce el tamaño que puede tener, por ejemplo una lista de apps.

En este caso, para los dispositivos móviles más pequeños se mostrará un máximo de 3 apps por fila, para los dispositivos móviles de tamaño mediano se mostrará un máximo de 4 apps por fila, para los dispositivos móviles o de escritorio grandes se mostrará un máximo de 6 apps por fila y finalmente para los dispositivos de cualquier tipo más grandes un máximo de 12 apps por fila

A continuación se muestra la implementación de unos elementos que se muestran de forma diferente dependiendo del tamaño del viewport del dispositivo, mediante media queries.

Las reglas CSS básicas para definir media queries son las siguientes:
Soy grande (lg)
Soy mediano (md)
Soy pequeño (sm)
Soy muy pequeño (xs)
