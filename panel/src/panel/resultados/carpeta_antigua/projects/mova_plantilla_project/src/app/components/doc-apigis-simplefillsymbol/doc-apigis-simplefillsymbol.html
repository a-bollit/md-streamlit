Generación de objeto apigis.SimpleFillSymbol

Se puede instanciar una simbología para entidades poligonales en 2D mediante la clase apigis.SimpleFillSymbol.

new ApiSimpleFillSymbol(args)

  • args <object > Objeto con todos los argumentos del contructor. Puede contener las siguientes propiedades:
    • type <string > Tipo de mapa sobre el que deseamos instanciar. Debe coincidir con alguno de los disponibles actualmente ([OL]).
    • properties <object > Propiedades asociadas al símbolo según el tipo de mapa (propiedad type anterior). Si type:'OL', estas propiedades pueden ser:
      • strokeColor <number[] | string > Color del contorno.
      • strokeWidth <number > Grosor del contorno.
      • lineDash <number[] > Patrón de línea punteada para el contorno.
      • lineDashOffset <number > Offset de línea punteada para el contorno.
      • fillColor <number[] | string > Color de relleno.
      • zIndex <number > Z index.
      • visible <boolean > Visibilidad.
      • text <object > Propiedades asociadas a un etiquetado. Puede contener las siguientes propiedades:
        • text <string > Texto de la etiqueta.
        • placement <object > Disposición de la etiqueta. Valores [point, line].
        • offsetX <number > Desplazamiento horizontal en píxeles de la etiqueta. Un valor positivo indica un desplazamiento a la derecha.
        • offsetY <number > Desplazamiento vertical en píxeles de la etiqueta. Un valor positivo indica un desplazamiento hacia abajo.
        • strokeColor <number[] | string> Color del contorno del texto.
        • strokeWidth <object >Grosor del contorno del texto.
        • fillColor <number[] | string > Color del texto.
        • font = '10px sans-serif' <string> Estilo css de la fuente del texto.
      • url <string > URL del SLD a aplicar.
      • headers <Headers > Cabeceras para la petición.
      • rules <string[] > Nombre de las reglas definidas en el SLD a aplicar.

Un símbolo puede alimentarse de las propiedades de estilo pasadas como parámetros o de un SLD pasado mediante una url.
