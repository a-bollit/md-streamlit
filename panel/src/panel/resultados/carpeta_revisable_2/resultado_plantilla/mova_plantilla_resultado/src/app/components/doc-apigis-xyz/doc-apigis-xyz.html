Consideraciones generales sobre Imágenes XYZ

Podemos generar capas de imágenes teleseladas XYZ a través de direcciones URL que incluyen un nivel de zoom y las coordenadas x/y de la cuadrícula de mosaico.

Para generar capas de Imágenes XYZ tileadas, debemos establecer typeLayer como 'Image' y subtypeLayer como 'XYZ' en la contrucción de un objeto de tipo apigis.Layer() como se explica en la documentación general sobre Servicios.

IMPORTANTE

Para poder visualizar una capa, deberá estar contenida dentro de un objeto de tipo apigis.Map.

Así mismo, un objeto de tipo apigis.Map deberá estar contenido dentro de un objeto de tipo apigis.Viewer.
