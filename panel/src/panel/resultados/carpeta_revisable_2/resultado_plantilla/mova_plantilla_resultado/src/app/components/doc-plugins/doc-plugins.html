Eliminar los plugins innecesarios

MOVA viene por defecto con una serie de plugins que pueden no ser necesarios para el cometido de nuestra aplicación. Así, lo óptimo sería eliminar los que no sean necesarios para que la aplicación no pida sus correspondientes permisos y el código sea más liviano

  • cordova-plugin-androidx

    No se puede eliminar porque se usa internamente en MOVA

    https://www.npmjs.com/package/cordova-plugin-androidx

    Versión del plugin: 2.0.0

  • cordova-plugin-androidx-adapter

    No se puede eliminar porque se usa internamente en MOVA

    https://www.npmjs.com/package/cordova-plugin-androidx-adapter

    Versión del plugin: 1.1.1

  • cordova-plugin-camera

    Si no se usa el componente mvQrScanner y la aplicación no usa la cámara eliminar este plugin

    https://www.npmjs.com/package/cordova-plugin-camera

    Versión del plugin: 4.1.0

    Este plugin pide permiso de escritura en almacenamiento exterior (android.permission.WRITE_EXTERNAL_STORAGE)

    Este plugin pide permiso de lectura en almacenamiento exterior (android.permission.READ_EXTERNAL_STORAGE)

    Este plugin pide permiso de uso de la cámara (android.permission.CAMERA)

    Este plugin pide permiso de uso de la cámara (iOS Camera)

    Este plugin pide permiso de uso de la galería (iOS PhotoLibrary)

    Este plugin pide permiso de localización (iOS Location)

    Este plugin pide permiso de escritura en la galería (iOS PhotoLibraryWrite)

  • cordova-plugin-console

    No se puede eliminar porque se usa internamente en MOVA

    https://www.npmjs.com/package/cordova-plugin-console

    Versión del plugin: 1.1.0

  • cordova-plugin-customurlscheme

    Este plugin se puede eliminar en caso de que no se use el broker con deep link o cualquier funcionalidad que use deep link

    https://www.npmjs.com/package/cordova-plugin-customurlscheme

    Versión del plugin: 4.4.0

  • cordova-plugin-device

    No se puede eliminar porque se usa internamente en MOVA

    https://www.npmjs.com/package/cordova-plugin-device

    Versión del plugin: 2.0.3

  • cordova-plugin-file

    Si no se usa la funcionalidad de pocket-card y no se usa la funcionalidad de cordova-plugin-file en la aplicación eliminar este plugin

    https://www.npmjs.com/package/cordova-plugin-file

    Versión del plugin: 6.0.2

    Este plugin pide permiso de escritura en almacenamiento exterior (android.permission.WRITE_EXTERNAL_STORAGE)

    Este plugin pide permiso de lectura en almacenamiento exterior (android.permission.READ_EXTERNAL_STORAGE)

  • cordova-plugin-file-transfer

    Si no se usa la funcionalidad de pocket-card eliminar este plugin

    https://www.npmjs.com/package/cordova-plugin-transfer

    Versión del plugin: 1.7.1

    Este plugin depende de cordova-plugin-file

    Este plugin pide permiso de escritura en almacenamiento exterior (android.permission.WRITE_EXTERNAL_STORAGE)

  • cordova-plugin-network-information

    No se puede eliminar porque se usa internamente en MOVA

    https://www.npmjs.com/package/cordova-plugin-network-information

    Versión del plugin: 2.0.2

    Este plugin pide permiso para acceder al estado de internet (android.permission.ACCESS_NETWORK_STATE)

  • cordova-plugin-safariviewcontroller

    No se puede eliminar porque se usa internamente en MOVA

    https://www.npmjs.com/package/cordova-plugin-safariviewcontroller

    Versión del plugin: 1.6.0

  • cordova-plugin-splashscreen

    No se puede eliminar porque se usa internamente en MOVA

    https://www.npmjs.com/package/cordova-plugin-splashscreen

    Versión del plugin: 6.0.0

  • cordova-plugin-statusbar

    No se puede eliminar porque se usa internamente en MOVA

    https://www.npmjs.com/package/cordova-plugin-statusbar

    Versión del plugin: 2.4.3

  • cordova-plugin-whitelist

    No se puede eliminar porque se usa internamente en MOVA

    https://www.npmjs.com/package/cordova-plugin-whitelist

    Versión del plugin: 1.3.4

  • cordova-support-google-services

    No se puede eliminar porque es necesario para phonegap-plugin-push

    https://www.npmjs.com/package/cordova-support-google-services

    Versión del plugin: 1.3.2

  • cordova.plugins.diagnostic

    No se puede eliminar porque se usa internamente en mova

    https://www.npmjs.com/package/phonegap-plugin-push

    Versión del plugin: 6.0.2

  • phonegap-plugin-multidex

    No se puede eliminar porque es necesario para phonegap-plugin-push

    https://www.npmjs.com/package/phonegap-plugin-multidex

    Versión del plugin: 1.0.0

  • phonegap-plugin-push

    Si no se usan notificaciones push eliminar este plugin

    https://www.npmjs.com/package/phonegap-plugin-push

    Versión del plugin: 2.3.0

    Este plugin depende de cordova-support-google-services

    Este plugin depende de phonegap-plugin-multidex

    Este plugin pide permiso para abrir conexiones (android.permission.INTERNET)

    Este plugin pide permiso para acceder al estado de la conexión (android.permission.ACCESS_NETWORK_STATE)

    Este plugin pide permiso para mantener la pantalla sin bloquear (android.permission.WAKE_LOCK)

    Este plugin pide permiso para usar el flash (android.permission.VIBRATE)
