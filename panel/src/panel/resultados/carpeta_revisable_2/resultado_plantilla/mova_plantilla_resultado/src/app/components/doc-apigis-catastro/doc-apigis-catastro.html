Documentación sobre Catastro

Cuando se genera un objeto de tipo apigis.Layer, puede heredar la clase Catastro tal y como se explica en la Herencia de Servidores. Esta clase permite heredarar dinámicamente el métodos de localización de coordenadas dentro de los servicios que ofrece Catastro.
