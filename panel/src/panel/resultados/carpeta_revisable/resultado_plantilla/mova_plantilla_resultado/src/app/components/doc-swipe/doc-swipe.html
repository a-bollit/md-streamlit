Información Información sobre swipe.

Hammerjs es una librera para la detección de gestos. Documentación.

Tiene varios gestos distintos, entre ellos:

  • Pan
  • Pinch
  • Press
  • Rotate
  • Swipe
  • Tap

En esta documentación solo expondremos ejemplos del swipe.

Configuración Configuración para establecer swipe en la aplicación

Para configurar esta funcionalidad debemos definir los siguientes elementos:

  • Crear un archivo ts que implementa la configuración hammer.config.js
  • Importar el archivo hammer.js en app.module.ts
